# Playwright TypeScript Automation Framework

A scalable end-to-end test automation framework for https://www.saucedemo.com/ built using Playwright and TypeScript.

The framework demonstrates modern automation practices including Page Object Model, UI and API automation, parallel execution, reusable fixtures, and CI integration.

---

## Tech Stack

- Playwright
- TypeScript
- Node.js
- Page Object Model
- Playwright Test Runner

---

## Project Structure

- `tests/`: Test scripts
- `pages/`: Page Object Model classes
- `fixtures/`: Custom fixtures
- `data/`: Test data (JSON)
- `.github/workflows/`: CI pipeline

---

## Features

- UI automation using Playwright
- Page Object Model architecture
- Parallel test execution
- Reusable fixtures
- Environment configuration support
- CI-ready structure

---

## How to Run
1. Install dependencies: `npm install`
2. Run tests: `npx playwright test`
3. View reports in `playwright-report/`

---
